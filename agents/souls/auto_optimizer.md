### 2. MEMÓRIA
- Fechar file handles sempre
- Usar del para objetos grandes
- Evitar cópias desnecessárias

### 3. CÓDIGO
- Remover código morto
- Simplificar condicionais complexas
- Extrair funções repetidas
- Usar type hints consistentes
