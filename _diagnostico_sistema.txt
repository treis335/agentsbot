 Ocorrências: 3.310 erros de polling + 4.895 Conflict

 - Supervisor (supervisor_ultra.py)

 1. Implementar lock de processo para Telegram (apenas 1 instância)

## PROBLEMA 2: OrganicMind — Debate Falha com Erro 'title'
 Severidade: ALTA
 Ocorrências: 168 (contínuo, desde 2026-05-30 13:39 até agora)
 Fonte: autonomous_log.md, autonomous_loop.py

 CAUSA:
 1. collective_debate() chama LLM que devolve JSON malformatado
 2. _extract_tasks() cria tarefas sem campo 'title' quando LLM falha
 3. _notify_debate() usa t['title'] em vez de t.get('title', '?')
 4. O except genérico em _run_organic_debate() chama _add_fallback_task()
 5. A tarefa de fallback é executada, gera outra falha, ciclo recomeça

 IMPACTO:
 - Loop infinito de falhas (168 ciclos perdidos)
 - Mesmas tarefas de fallback repetidas infinitamente
 - Sistema preso num ciclo vicioso

 CORREÇÃO IMPLEMENTADA: ✅
 1. _notify_debate(): t.get('title', '?') → t.get('title', t.get('description', '?'))
 2. _extract_tasks(): Garantir que title e description existem sempre
 3. Adicionado stack trace ao log para debugging futuro
